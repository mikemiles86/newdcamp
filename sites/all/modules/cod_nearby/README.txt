COD Nearby
----------


Requirements
--------------------------------------------------------------------------------
COD Nearby 7.x-1.x requires the following:

* COD: https://www.drupal.org/project/cod
* COD Support: https://www.drupal.org/project/cod_support
* Addressfield: https://www.drupal.org/project/addressfield
* Geocoder: https://www.drupal.org/project/geocoder
* Geofield: https://www.drupal.org/project/geofield


Features
--------------------------------------------------------------------------------


Standard usage scenario
--------------------------------------------------------------------------------
1. Install the module.
2. Assign the permissions as necessary for the "Nearby" content type.
3. Go to the Geofield Map settings page (admin/config/content/geofield_map) and
   add a Google Maps API key.
